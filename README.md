# IconCreator

Create app icons on Playground.

```
let c = IconCreator()
IconCreator.loadFont("devicons.ttf")
c.fontName = "icomoon"
c.string = "\u{e655}"
c.backgroundColor = UIColor.orangeColor()
c.preview()
```
